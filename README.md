## Project Reason
I create this personal page to practice what I learned in the HTML training videos.  I tried to work with Bootstrap layouts on the two pages I created.  I added accessbility around the form I created on the contact page and a couple breakpoints for desktop and mobile browsers.

I tried to work on my HTML structuring based off the videos and training we have done in the past couple weeks.  Making sure the headers and footers came out the way I wanted to and that the form I created was organized within several <div> tags.  I also tried to make sure that the code was easily readable.  I spent a lot on time on the style sheet to make sure it was properly organized and the comments reflected what exactly was being styled and why.

For the form each label uses the for attribute that matches the id of its corresponding input or textarea. This ensures screen readers and assistive technologies can properly associate labels with their form fields.  

I tried to page the pages equally assessible and usable by Desktop and Mobile browsers.  I wouldn't say I tried to prefer one or the other.  I wanted to make sure that I did the best I could making the sites usable by most broswers.
